Linking a Stylesheet

When linking a stylesheet the first thing you need is a stylesheet! Here we are using a provided Pure.CSS link. Inside of the head tag you will create another tag, the link tag. In the link tag you will add two attributes. One: the 'rel' or relative attribute that tells us what the relationship of this new file is to our html. Two: you'll add 'href' which is a hyperlink reference to the file containing the styling data.
